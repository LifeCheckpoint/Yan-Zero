# Chat-Vision

GPT Powered 的插件。

## Chat

基于 GPT-4o 的聊天功能

### chat

> /chat [Message]

与 GPT-4o 聊天。

| 参数 | 说明 |
| ---- | ---- |
| Message | 聊天内容 |

#### 一些参数

每次回复都会带上上下文……嗯，每人每天也就能用 0.1$，所以如果不及时清理，回复很长的花，三四次就达到限制了。

#### 示例

```
/chat 你好
```

```
你好，我是 GPT-4o，很高兴和你聊天。
```

```
/chat 这上面写了什么？[Image]
```

```
上面写的是“你好，我是 GPT-4o，很高兴和你聊天。”
```

### reset

> /reset

重置上下文。可以避免幻觉，以及太烧钱等。

#### 示例

```
/reset
```

### draw

> /draw [Prompt]

生成图片。用 Dall-e-3 生成，随时都可能不可用

| 参数 | 说明 |
| ---- | ---- |
| Prompt | 图片描述 |

#### 示例

```
/draw 猫
```
